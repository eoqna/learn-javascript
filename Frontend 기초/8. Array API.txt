Array API


map() : 배열 내의 모든 요소에 대하여 주어진 함수를 호출한 결과를 모아 새로운 배열로 반환해주는 메소드


some() : 배열 안의 어떤 요소라도 주어진 조건을 통과하는지 테스트하는 메소드 요소가 하나라도 true이면 true를 반환한다.


every() : 배열 안의 모든 요소가 주어진 조건을 통과하는지 테스트하는 메소드

filter() : 주어진 함수의 테스트를 통과하는 모든 요소를 모아 새로운 배열로 반환해주는 메소드

reduce() : 배열의 각 요소에 대해 주어진 리듀서(reducer) 함수를 실행하고, 하나의 결과값을 반환해주는 메소드

* 리듀서 함수 : 누적 값(acc), 현재 값(cur), 현재 인덱스(idx), 원본 배열(src)
