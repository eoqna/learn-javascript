API란

Application Programming Interface

응용프로그램에서 사용할 수 있도록 운영체제나 프로그래밍 언어가 제공하는 기능을 제어할 수 있게 만든 인터페이스

애플리케이션에서 데이터를 읽거나 쓰기 위해 사용하는 인터페이스

응용프로그램 : 운영체제에서 실행되는 모든 소프트웨어, 애플리케이션

종류

private API, Public API(Open API)

인터페이스란

상호간에 소통을 위해 만들어진 접점

ex)
어딘가를 가기 위해 차를 타려고 한다면 말로 '열려라'라고 아무리 얘기해도 차 문은 열리지않는다.

하지만 스마트키를 사용하여 '열림'버튼을 누르면 차 문이 열린다. (여기서 스마트키를 interface라고 할 수 있다.)



UI 란

User Interface, 사용자가 소통하기 위한 접점을 뜻한다.



HTTP API

보통 미팅할때 API, OPEN API를 말하는 경우 HTTP API에 해당한다.

인터넷 상에서 데이터를 주고받을 때 사용하는 Protocol
